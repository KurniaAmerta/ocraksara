Traning
- Traning merupakan bagian admin untuk input data latih
- Diakses pada: http://localhost/ocr_aksara/admin
- step by step:
1) input label (misal: "ha")
2) input gambar (misal: Data0.JPG, sudah tersedia pada folder ini, hanya boleh jpg untuk menghindari warna transparan dan hanya file gambar)
3) button proses untuk menampilkan hasil untuk saat ini baru preprocessing dan ekstraksi fitur
4) button unduh untuk melihat data hasil ektraksi fitur diagonal

Testing
- Testing merupakan bagian player atau member untuk input data testing
- Diakses pada: http://localhost/ocr_aksara/
- step by step:
1) saat ini baru tersedia menu kuis dummy > button mulai > button kuis atau bisa langsung akses pada: http://localhost/ocr_aksara/memberKuis
2) button hapus untuk hapus data
3) button selesai untuk menampilkan hasil untuk saat ini baru preprocessing dan ekstraksi fitur
4) button unduh untuk melihat data hasil ektraksi fitur diagonal
5) button back untuk menu sebelumnya
